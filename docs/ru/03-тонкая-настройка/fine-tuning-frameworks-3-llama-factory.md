Для многих первое знакомство с тонкой настройкой больших моделей начинается с LoRA в LLaMA-Factory. Это похоже на то, как раньше дата-сайентисты начинали с Kaggle-задачи Titanic survival prediction: LoRA в LLaMA-Factory — очень хорошая отправная точка.

## 1. Web-UI

LLaMA Factory предоставляет удобный визуальный интерфейс (WebUI), который позволяет пользователям выполнять тонкую настройку больших языковых моделей без написания кода.

**Обзор интерфейса**:
   - WebUI в LLaMA Factory в основном состоит из четырех разделов: обучение, оценка и предсказание, диалог, экспорт.

**Интерфейс обучения**:
   - В интерфейсе обучения пользователь может указать имя и путь модели, стадию обучения, метод тонкой настройки, обучающий датасет, learning rate, число эпох и другие параметры обучения, а затем запустить обучение.
   - Поддерживается возобновление после прерывания; checkpoint адаптера сохраняется в каталоге `output_dir`.

**Интерфейсы оценки, предсказания и диалога**:
   - После завершения обучения пользователь может в интерфейсе оценки и предсказания указать пути к модели и адаптеру и провести оценку на заданном датасете.
   - Пользователь также может общаться с моделью через интерфейс диалога и наблюдать качество работы модели.

**Интерфейс экспорта**:
   - Если пользователь доволен качеством модели и хочет экспортировать ее, в интерфейсе экспорта можно указать модель, адаптер, размер блоков, уровень квантизации экспорта, калибровочный датасет, устройство экспорта, каталог экспорта и другие параметры, а затем нажать кнопку экспорта.

**Шаги использования**:
   - Пользователь может открыть в браузере `http://localhost:7860`, перейти в Web-интерфейс, выбрать модель, настроить параметры и отслеживать прогресс задачи обучения.
   - Поддерживается переключение языка на китайский, что удобно для пользователей из Китая.

**Поддерживаемые модели и датасеты**:
   - LLaMA Factory поддерживает множество предобученных моделей, например LLaMA, BLOOM, Mistral и другие, а также умеет автоматически скачивать и кэшировать ресурсы моделей и датасетов из сообщества ModelScope.

**Методы тонкой настройки**:
   - Доступны разные методы тонкой настройки, включая full fine-tuning, freeze fine-tuning, LoRA fine-tuning и другие.

**Настройка параметров**:
   - WebUI позволяет оптимизировать модель с помощью простого drag-and-drop и настройки параметров, поэтому даже пользователи без опыта программирования могут легко участвовать в процессе оптимизации модели.

## 2. За пределами LoRA

LLaMA-Factory предоставляет набор продвинутых возможностей для поддержки сложных сценариев тонкой настройки и развертывания больших моделей:

**Совместимость со многими моделями**: LLaMA-Factory поддерживает разные большие языковые модели, например LLaMA, BLOOM, Mistral, Baichuan, Qwen, ChatGLM и другие;

**Интеграция обучающих алгоритмов**: фреймворк интегрирует разные обучающие алгоритмы, включая incremental pre-training, instruction supervised fine-tuning, reward model training, PPO training, DPO training, KTO training, ORPO training и другие;

**Точность вычислений и алгоритмы оптимизации**: доступны разные варианты точности вычислений, например 32-битная full fine-tuning, 16-битная freeze fine-tuning, 16-битная LoRA fine-tuning, а также 2/4/8-битная QLoRA fine-tuning на основе AQLM/AWQ/GPTQ/LLM.int8. Также поддерживаются продвинутые алгоритмы GaLore, DoRA, LongLoRA, LLaMA Pro, LoRA+, LoftQ и Agent fine-tuning;

**Поддержка inference engines**: поддерживаются inference engines Transformers и vLLM, что дает пользователям гибкие варианты inference;

**Интеграция экспериментальных панелей**: есть интеграция с LlamaBoard, TensorBoard, Wandb, MLflow и другими инструментами экспериментальных панелей, что удобно для мониторинга и анализа процесса обучения;

**Функции API Server**: LLaMA-Factory поддерживает запуск API Server, позволяя пользователям вызывать модель через API-интерфейс и упрощая интеграцию и применение модели;

**Benchmark для оценки больших моделей**: предоставляются популярные benchmark-инструменты для оценки больших моделей, которые помогают пользователям измерять качество модели;

**Установка через Docker и адаптация под Huawei NPU**: поддерживаются установка через Docker и работа на Huawei NPU, что повышает переносимость фреймворка и совместимость с аппаратным обеспечением;

**Технологии квантизации**: поддерживаются PTQ, QAT, AQLM, OFTQ и другие технологии квантизации, оптимизирующие эффективность и производительность развертывания модели;

Эти продвинутые возможности позволяют LLaMA-Factory не только поддерживать базовую тонкую настройку моделей, но и закрывать более сложные исследовательские и прикладные потребности, предоставляя пользователям мощный и гибкий набор инструментов.
