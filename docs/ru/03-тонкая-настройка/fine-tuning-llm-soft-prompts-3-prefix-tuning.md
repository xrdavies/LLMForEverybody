LoRA — не единственный вариант: секреты fine-tuning больших моделей через Soft Prompts, часть 3: Prefix-Tuning

В том же 2021 году, что и Prompt Tuning, был предложен Prefix-Tuning. Он добавляет обучаемые векторы подсказок в позицию префикса на входе модели. Преимущество этого метода в том, что он позволяет задавать разные подсказки для разных задач без изменения структуры модели.

## Технический разбор

![alt text](../../../03-第三章-微调/assest/大模型微调之Soft prompts（三）Prefix-tuning/0.png)

Prefix Tuning — это параметрически эффективная техника тонкой настройки для задач генерации естественного языка. Ее ключевая идея состоит в том, чтобы добавить перед входной последовательностью модели набор непрерывных векторов, специфичных для задачи. Эти векторы называются префиксом (prefix). Префиксные векторы обучаются, а остальные параметры предобученной языковой модели (PLM) остаются замороженными.

В Prefix Tuning вход модели включает не только исходный вход задачи, но и эти префиксные векторы. В авторегрессионных языковых моделях, например GPT-2, префикс добавляется в начало входной последовательности, формируя новую входную последовательность. В моделях архитектуры encoder-decoder, например BART, префикс добавляется не только в начало входной последовательности, но и на вход декодера. Благодаря этому при обработке входной последовательности вход каждого слоя содержит дополнительные префиксные векторы, что помогает адаптировать модель к downstream-задаче.

Одно из ключевых преимуществ Prefix Tuning — параметрическая эффективность. Метод обновляет только небольшую часть параметров, то есть префиксные векторы, и не требует обновлять параметры всей модели. Это значительно снижает требования к вычислительным ресурсам и хранению. Кроме того, поскольку обновляются только префиксные векторы, Prefix Tuning проще адаптировать к нескольким задачам: не нужно обучать и хранить полную копию модели для каждой задачи.

На практике Prefix Tuning показал эффективность в разных задачах обработки естественного языка, включая генерацию текста и суммаризацию. Добавляя обучаемые префиксные векторы ко входу модели, он позволяет модели лучше адаптироваться к конкретным downstream-задачам без изменения исходных параметров.

## Интуитивное объяснение

Главное различие между Prefix-Tuning и Prompt Tuning в том, что параметры префикса в Prefix-Tuning вставляются во все слои модели, тогда как Prompt Tuning добавляет параметры подсказки только на embedding-слой модели.
