## 1. GELU

![alt text](../../../01-第一章-预训练/assest/神经网络的激活函数/10.png)

GELU (Gaussian Error Linear Unit, линейный блок с гауссовой ошибкой) — широко применяемая в глубоком обучении функция активации. GELU сглаживает вход с помощью гауссовой функции ошибок, то есть функции распределения стандартного нормального распределения, тем самым повышая качество модели. GELU хорошо показывает себя во многих задачах, особенно в обработке естественного языка (NLP) и компьютерном зрении.

### 1.1 Математическое определение

Математическое выражение функции GELU:

$$\text{GELU}(x) = x \cdot \Phi(x)$$

где:
- $x$ — вход.
- $\Phi(x)$ — функция распределения стандартного нормального распределения, определяемая как:
$\Phi(x) = \frac{1}{2} \left( 1 + \text{erf}\left( \frac{x}{\sqrt{2}} \right) \right)$

где $\text{erf}(x)$ — функция ошибок.

### 1.2 Приближенное выражение

Чтобы упростить вычисления, для функции GELU обычно используют следующее приближенное выражение:

$$\text{GELU}(x) \approx 0.5 \cdot x \cdot \left( 1 + \tanh\left( \sqrt{\frac{2}{\pi}} \left( x + 0.044715 x^3 \right) \right) \right)$$

Такое приближенное вычисление очень распространено на практике, потому что оно эффективнее с точки зрения вычислений.

### 1.3 Ключевые свойства

1. **Гладкость**: функция GELU непрерывна и гладка, что помогает повысить стабильность модели и скорость сходимости.
2. **Нелинейность**: GELU сочетает линейное и нелинейное преобразования, позволяя модели изучать сложные паттерны.
3. **Вероятностная интерпретация**: GELU сглаживает вход с помощью гауссовой функции ошибок и поэтому имеет вероятностную интерпретацию: чем больше входное значение, тем выше вероятность его прохождения.

### 1.4 Когда была предложена

Функция активации GELU была предложена в 2016 году Dan Hendrycks и Kevin Gimpel в статье «Gaussian Error Linear Units (GELUs)».

### Какие проблемы решает

1. **Гладкая активация**: GELU вводит гауссову функцию ошибок, благодаря чему активация остается гладкой и на положительном, и на отрицательном участках входа, повышая стабильность модели.
2. **Усиление нелинейности**: GELU сочетает линейное преобразование и нелинейную активацию, усиливая нелинейные свойства модели.
3. **Вероятностная интерпретация**: GELU имеет вероятностную интерпретацию: чем больше входное значение, тем выше вероятность его прохождения. Это лучше имитирует процесс активации нейрона.


### Пример

Ниже приведен простой пример на Python, показывающий, как вычислить функцию GELU:

```python
from scipy.special import erf

# 定义GELU函数
def gelu(x):
    return 0.5 * x * (1 + erf(x / np.sqrt(2)))
```

## Mish

![alt text](../../../01-第一章-预训练/assest/神经网络的激活函数/11.png)

Mish — нелинейная функция активации, используемая в глубоком обучении. Она была предложена Diganta Misra в 2019 году. Mish усиливает производительность модели с помощью гладкого нелинейного преобразования и хорошо показывает себя во многих задачах, включая компьютерное зрение и обработку естественного языка.

### Математическое определение

Математическое выражение функции Mish:

$$\text{Mish}(x) = x \cdot \tanh(\text{softplus}(x))$$

где:
- $\text{softplus}(x) = \ln(1 + e^x)$ — функция Softplus.
- $\tanh(x)$ — гиперболический тангенс.

### Ключевые свойства

1. **Гладкость**: функция Mish непрерывна и гладка, что помогает повысить стабильность модели и скорость сходимости.
2. **Немонотонность**: функция Mish немонотонна: на одних участках она возрастает, а на других убывает. Это свойство позволяет Mish улавливать более сложные паттерны.
3. **Мягкая отрицательная область**: в отличие от ReLU, функция Mish сохраняет ненулевые отрицательные значения, что помогает потоку градиентов и предотвращает проблему исчезновения градиента.
4. **Приближение к ReLU**: при больших входных значениях функция Mish приближается к ReLU.

### Когда была предложена

Функция активации Mish была предложена в 2019 году Diganta Misra в статье «Mish: A Self Regularized Non-Monotonic Neural Activation Function».


### Какие проблемы решает

1. **Гладкая активация**: Mish вводит Softplus и гиперболический тангенс, благодаря чему активация остается гладкой и на положительном, и на отрицательном участках входа, повышая стабильность модели.
2. **Немонотонность**: немонотонность Mish позволяет ей улавливать более сложные паттерны и делает ее выразительнее монотонных функций активации вроде ReLU.
3. **Поток градиентов**: Mish сохраняет мягкую отрицательную область, что помогает потоку градиентов и предотвращает проблему исчезновения градиента.


### Пример

Ниже приведен простой пример на Python, показывающий, как вычислить функцию Mish:

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义Softplus函数
def softplus(x):
    return np.log1p(np.exp(x))

# 定义Mish函数
def mish(x):
    return x * np.tanh(softplus(x))

```
