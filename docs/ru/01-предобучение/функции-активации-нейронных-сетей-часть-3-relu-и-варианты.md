Функции активации нейронных сетей (часть 3): ReLU и ее варианты Leaky-ReLU, PReLU

В этой статье мы рассмотрим ReLU, одну из важных причин успеха глубокого обучения, и ее варианты. Их широкое применение в нейронных сетях имеет большое значение для повышения качества моделей и ускорения обучения.

## 1. Функция ReLU

![alt text](../../../01-第一章-预训练/assest/神经网络的激活函数/3.png)

### 1.1 Определение

ReLU (Rectified Linear Unit, выпрямленный линейный блок) — одна из самых распространенных функций активации в современном глубоком обучении. Ее математическое выражение:

$$\text{ReLU}(x) = \max(0, x)$$

### 1.2 Ключевые свойства

1. **Нелинейность**: хотя ReLU линейна на положительном участке, она вносит нелинейные свойства, позволяя нейронной сети изучать сложные паттерны.
2. **Разреженная активация**: ReLU превращает отрицательную часть входа в ноль. Это означает, что на практике активации нейронов разрежены, то есть активна только часть нейронов, что помогает повышать эффективность и качество модели.
3. **Простота вычислений**: ReLU вычисляется очень просто: достаточно сравнить входное значение с нулем. Объем вычислений мал, что помогает ускорить обучение.


### 1.3 Время появления

В 2010 году Vinod Nair и Geoffrey Hinton в статье «Rectified Linear Units Improve Restricted Boltzmann Machines» показали эффективность ReLU в глубоких нейронных сетях. С тех пор ReLU стала одной из самых популярных функций активации в глубоком обучении.

### 1.4 Преимущества и недостатки

***Преимущества***

1. **Высокая вычислительная эффективность**: ReLU проста в вычислении и может существенно ускорять обучение нейронных сетей.
2. **Меньшая проблема исчезающего градиента**: по сравнению с Sigmoid и Tanh, градиент ReLU на положительном участке равен константе 1. Это помогает ослабить проблему исчезающего градиента и упрощает обучение глубоких сетей.

***Недостатки***

1. **Проблема Dying ReLU**: в процессе обучения некоторые нейроны могут никогда не активироваться, то есть их вход всегда остается отрицательным. В результате эти нейроны не вносят вклад на протяжении всего обучения. Чтобы решить эту проблему, исследователи предложили такие варианты, как Leaky ReLU и Parametric ReLU.
2. **Асимметрия**: в отрицательной области выход ReLU всегда равен нулю, что в некоторых случаях может ухудшать качество модели.


## 2. Функция Leaky ReLU
Leaky ReLU (Leaky Rectified Linear Unit, выпрямленный линейный блок с утечкой) — вариант функции активации ReLU, предназначенный для решения проблемы «Dying ReLU». Проблема Dying ReLU означает, что в процессе обучения некоторые нейроны могут никогда не активироваться, то есть их вход всегда остается отрицательным, и из-за этого они не вносят вклад на протяжении всего обучения.

![alt text](../../../01-第一章-预训练/assest/神经网络的激活函数/4.png)

### 2.1 Математическое определение

Математическое выражение Leaky ReLU:

$$\text{Leaky ReLU}(x) = \begin{cases}
x & \text{if } x \geq 0 \\
\alpha x & \text{if } x < 0
\end{cases}$$

Здесь $\alpha$ — малое положительное число, обычно около 0.01.

### 2.2 Ключевые свойства

1. **Нелинейность**: как и ReLU, Leaky ReLU вносит нелинейные свойства, позволяя нейронной сети изучать сложные паттерны.
2. **Разреженная активация**: хотя Leaky ReLU в отрицательной области не становится полностью равной нулю, она все равно сохраняет некоторую разреженность, помогая повышать эффективность и качество модели.
3. **Простота вычислений**: Leaky ReLU также вычисляется просто: в отрицательной области достаточно умножить значение на небольшую константу $\alpha$.
4. **Избежание проблемы Dying ReLU**: благодаря введению небольшого наклона $\alpha$ в отрицательной области Leaky ReLU гарантирует наличие градиента у всех нейронов и тем самым избегает проблемы Dying ReLU.


### 2.3 Время появления

В 2013 году Leaky ReLU впервые появилась в статье «Rectifier Nonlinearities Improve Neural Network Acoustic Models», написанной Andrew L. Maas, Awni Y. Hannun и Andrew Y. Ng.


## Функция PReLU

![alt text](../../../01-第一章-预训练/assest/神经网络的激活函数/5.png)

PReLU (Parametric Rectified Linear Unit, параметрический выпрямленный линейный блок) — еще один вариант функции активации ReLU. Он вводит обучаемый параметр для управления наклоном в отрицательной области. PReLU предназначена для дальнейшего улучшения качества ReLU и ее вариантов, таких как Leaky ReLU.

### 3.1 Математическое определение

Математическое выражение PReLU:

$$\text{PReLU}(x) = \begin{cases}
x & \text{if } x \geq 0 \\
\alpha x & \text{if } x < 0
\end{cases}$$

Здесь $\alpha$ — обучаемый параметр, а не фиксированная константа.

### 3.2 Ключевые свойства

1. **Нелинейность**: как и ReLU и Leaky ReLU, PReLU вносит нелинейные свойства, позволяя нейронной сети изучать сложные паттерны.
2. **Разреженная активация**: хотя PReLU в отрицательной области не становится полностью равной нулю, она все равно сохраняет некоторую разреженность, помогая повышать эффективность и качество модели.
3. **Обучаемый параметр**: главная особенность PReLU в том, что наклон отрицательной области $\alpha$ является обучаемым. Это означает, что модель может автоматически настраивать этот параметр по данным и находить оптимальный наклон отрицательной области в процессе обучения.
4. **Избежание проблемы Dying ReLU**: благодаря введению обучаемого параметра наклона $\alpha$ PReLU гарантирует наличие градиента у всех нейронов и эффективно избегает проблемы Dying ReLU.


### 3.3 Время появления

PReLU предложили Kaiming He, Xiangyu Zhang, Shaoqing Ren и Jian Sun в статье 2015 года «Delving Deep into Rectifiers: Surpassing Human-Level Performance on ImageNet Classification».
