## 1. Появление алгоритма RMSProp
Алгоритм RMSProp (Root Mean Square Propagation) был предложен Geoffrey Hinton в его курсе Coursera «Neural Networks for Machine Learning» [1](#refer-anchor-8), впервые опубликованном в 2012 году. RMSProp - это метод оптимизации с адаптивным learning rate. Он использует экспоненциальное скользящее среднее квадратов градиентов, чтобы настраивать learning rate для каждого параметра, ускорять обучение и уменьшать колебания в процессе тренировки. Такой метод особенно хорошо подходит для задач невыпуклой оптимизации и широко применяется в глубоком обучении.

## 2. Принцип работы алгоритма RMSProp

RMSProp (Root Mean Square Propagation) - это алгоритм оптимизации с адаптивным learning rate, предложенный для решения проблемы убывания learning rate в AdaGrad. RMSProp использует экспоненциально взвешенное скользящее среднее, чтобы настраивать learning rate для каждого параметра и делать эту настройку более плавной.

Правила обновления RMSProp выглядят так:
1. Инициализировать параметры $\theta$, задать learning rate $\eta$, коэффициент затухания $\rho$ (обычно 0.9), а также малую константу численной стабильности $\epsilon$ (обычно $1e-8$);
2. На каждой итерации вычислить градиент $g$ параметров $\theta$;
3. Обновить экспоненциально взвешенное скользящее среднее накопленных квадратов градиентов $r$:
   $r = \rho \cdot r + (1 - \rho) \cdot g^2$
4. Вычислить обновление параметров:
   $\Delta\theta = \frac{\eta}{\sqrt{r + \epsilon}} \cdot g$
5. Обновить параметры $ \theta $:
   $\theta = \theta - \Delta\theta$

## 3. Основные особенности алгоритма RMSProp

Преимущества алгоритма RMSProp включают:
- Адаптивная настройка learning rate, без необходимости настраивать learning rate вручную;
- Хорошо подходит для нестационарных целевых функций и рекуррентных нейронных сетей (RNN);
- Может смягчать проблему исчезающих или взрывающихся градиентов.

Недостатки алгоритма RMSProp включают:
- Нужно настраивать новый гиперпараметр - скорость затухания $\rho$;
- Алгоритм по-прежнему зависит от глобального learning rate $\eta$; если он выбран неудачно, качество обучения модели может оказаться плохим.

На практике рекомендуется начинать с небольшого глобального learning rate и постепенно увеличивать его, чтобы найти лучшую производительность. Также можно пробовать разные значения скорости затухания $\rho$, чтобы подобрать настройку, наиболее подходящую для модели.

## 4. Различия между RMSProp и AdaGrad

Главное улучшение заключается в способе обработки накопленных градиентов:

- AdaGrad: накапливает квадраты всех прошлых градиентов (без фактора забывания). Это означает, что он непрерывно накапливает информацию о градиентах, из-за чего learning rate со временем постепенно уменьшается и на поздних этапах может стать слишком маленьким для эффективного продолжения обновлений;

- RMSProp: использует экспоненциально взвешенное среднее для накопления квадратов прошлых градиентов (с фактором забывания). Такой подход дает больший вес последним градиентам и постепенно «забывает» старые, тем самым избегая слишком быстрого уменьшения learning rate.
