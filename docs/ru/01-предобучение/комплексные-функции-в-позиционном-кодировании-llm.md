## 2. Что такое комплексное число
Комплексное число - это базовое математическое понятие, которое расширяет систему действительных чисел и позволяет выполнять более широкий набор математических операций. Комплексное число состоит из двух частей: действительной и мнимой. Общий вид комплексного числа можно записать так:

$$ a + bi$$

где $a$ - действительная часть комплексного числа, $b$ - мнимая часть, а $i$ - мнимая единица, для которой выполняется $ i^2 = -1 $.

У комплексных чисел есть две основные особенности:

1. **Действительное число**: когда мнимая часть $b$ равна 0, комплексное число вырождается в действительное.
2. **Мнимое число**: когда действительная часть $a$ равна 0, а мнимая часть $b$ ненулевая, комплексное число называют чисто мнимым.

С комплексными числами можно выполнять базовые арифметические операции: сложение, вычитание, умножение и деление. Эти операции подчиняются определенным правилам. Например, при сложении двух комплексных чисел отдельно складываются их действительные и мнимые части:

$$ (a + bi) + (c + di) = (a + c) + (b + d)i $$

Умножение комплексных чисел немного сложнее: здесь используется распределительный закон и свойство мнимой единицы $i$:

$$(a + bi)(c + di) = ac + adi + bci + bdi^2
 = (ac - bd) + (ad + bc)i $$

Деление комплексных чисел связано с рационализацией знаменателя; обычно это делают умножением на сопряженное комплексное число.

Комплексные числа широко применяются в математике, физике, инженерии и других областях. Например, в обработке сигналов, квантовой механике и электротехнике комплексные числа дают мощный инструмент для описания периодических явлений и вращений.

## 3. Функции комплексной переменной
Функции комплексной переменной - важный раздел математики. Он изучает функции над областью комплексных чисел, где и аргумент, и значение функции являются комплексными числами. Теория функций комплексной переменной применяется во многих инженерных и научных областях. Вот некоторые специальности, где обычно изучают комплексный анализ:

![alt text](../../../01-第一章-预训练/assest/复变函数在大模型位置编码中的应用/7.png)

- Математика: функции комплексной переменной - один из обязательных курсов для студентов-математиков, потому что это продолжение и углубление математического анализа.
- Электротехника: в обработке сигналов и системном анализе функции комплексной переменной используются для анализа цепей переменного тока.
- Электронная инженерия: методы в комплексной области применяются при проектировании и анализе систем управления.
- Физика: в квантовой механике и электромагнетизме теория функций комплексной переменной используется для решения волновых уравнений и задач о потенциалах.

- Компьютерные науки и инженерия: функции комплексной переменной применяются в разработке алгоритмов, обработке изображений и алгоритмах обработки сигналов.

- Аэрокосмическая инженерия: в гидродинамике и анализе систем управления функции комплексной переменной используются для математического моделирования.

- Машиностроение: теория функций комплексной переменной помогает находить решения в задачах анализа вибраций и теплопроводности.

- Гражданское строительство: в структурном анализе и сейсмической инженерии функции комплексной переменной используются для решения некоторых динамических задач.

- Биология и биомедицинская инженерия: в обработке биосигналов и биофизическом моделировании функции комплексной переменной помогают анализировать и понимать динамическое поведение биологических систем.

- Финансовая математика и экономика: в некоторых продвинутых экономических моделях и при оценке финансовых инструментов теория функций комплексной переменной может давать аналитические инструменты.

- Теория управления: в анализе устойчивости систем и проектировании контроллеров функции комплексной переменной являются одним из базовых инструментов.

Курс функций комплексной переменной обычно включает понятия аналитических функций, комплексных интегралов, разложения в ряды, теоремы о вычетах, сопряженных гармонических функций и т. д.

В машинном обучении комплексные числа применяются сравнительно редко, но в некоторых специальных областях, таких как обработка сигналов, обработка изображений и распознавание образов, свойства комплексных чисел и правила операций с ними могут дать полезные инструменты и техники.

Через десять лет после выпуска я снова ощутил привлекательность комплексных чисел - на этот раз в позиционном кодировании больших языковых моделей. Математические свойства комплексных чисел и правила операций с ними дали Transformer новый способ позиционного кодирования. Этот метод называется RoPE (Rotary Position Embedding).


## 4. Позиционное кодирование и RoPE


Позиционное кодирование (Positional Encoding) - это техника, которая при обработке последовательностей передает модели информацию о положении каждого элемента в последовательности.

В обработке естественного языка (NLP), особенно при использовании моделей Transformer, позиционное кодирование особенно важно, потому что сам Transformer не содержит встроенного механизма обработки порядка элементов в последовательности.

Главная цель позиционного кодирования - позволить модели различать порядок слов во входной последовательности и тем самым лучше понимать структуру и смысл предложения.

![alt text](../../../01-第一章-预训练/assest/复变函数在大模型位置编码中的应用/1.PNG)

**Вращательное позиционное кодирование RoPE**

RoPE предлагает использовать информацию об относительных позициях между token. Предполагается, что операция скалярного произведения между вектором query $q_m$ и вектором key $k_n$ может быть выражена некоторой функцией $g$. На вход этой функции $g$ подаются эмбеддинги слов $x_m$, $x_n$ и их относительная позиция $m-n$:

![alt text](../../../01-第一章-预训练/assest/复变函数在大模型位置编码中的应用/2.png)

Смелая гипотеза, аккуратная проверка. Теперь наша цель - найти подходящую функцию $g$, чтобы $g(x_m, x_n, m-n)$ могла улавливать информацию об относительном положении между векторами слов.

RoPE утверждает: если векторы слов двумерны, можно превратить плоскость в комплексную плоскость. Если определить функцию $f$ следующим образом, то можно найти соответствующую $g$:
