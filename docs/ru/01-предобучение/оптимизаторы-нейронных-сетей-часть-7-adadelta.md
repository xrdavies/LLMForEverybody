## 1. Появление алгоритма AdaDelta
Алгоритм AdaDelta был предложен Matthew D. Zeiler в 2012 году. Подробное описание и принципы этого алгоритма можно найти в статье «ADADELTA: An Adaptive Learning Rate Method» [1](#refer-anchor-7). AdaDelta предназначен для решения проблемы монотонного уменьшения learning rate в алгоритме AdaGrad: он ограничивает размер окна накопленных градиентов и тем самым адаптирует learning rate. Благодаря этому во время обучения алгоритм может адаптивно настраивать learning rate для каждого параметра без ручной настройки. Такой подход показывает достаточно высокую устойчивость к шумной информации о градиентах, разным архитектурам моделей, различным типам данных и выбору гиперпараметров.


## 2. Принцип работы алгоритма AdaDelta

AdaDelta - это алгоритм оптимизации с адаптивным learning rate, который решает проблему, возникающую в AdaGrad из-за убывающего learning rate. Алгоритм AdaDelta ограничивает размер окна накопленных градиентов и не требует задания глобального learning rate, поскольку адаптивно настраивает learning rate на основе предыдущих обновлений параметров.

Правила обновления AdaDelta выглядят так:
1. Инициализировать две переменные состояния: экспоненциально взвешенное среднее накопленных квадратов градиентов `s` и экспоненциально взвешенное среднее накопленных обновлений `delta`.
2. На каждой итерации вычислить градиент `g`.
3. Обновить экспоненциально взвешенное среднее накопленных квадратов градиентов `s`:
   $s = \rho \cdot s + (1 - \rho) \cdot g^2$
4. Вычислить обновление параметра `delta_p`:
   $\delta_p = -\frac{\sqrt{\delta + \epsilon}}{\sqrt{s + \epsilon}} \cdot g$
5. Обновить параметр `w`:
   $w = w + \delta_p$
6. Обновить экспоненциально взвешенное среднее накопленных обновлений `delta`:
   $\delta = \rho \cdot \delta + (1 - \rho) \cdot \delta_p^2$

Здесь `ρ` - коэффициент для вычисления экспоненциально взвешенного среднего квадратов градиентов (обычно задается равным 0.9), а `ε` - очень малое число (например, `1e-6`), которое повышает численную стабильность вычислений.

## 3. Основные особенности алгоритма AdaDelta

Преимущества алгоритма AdaDelta включают:
- Адаптивный learning rate, который не нужно настраивать вручную.
- Хорошо подходит для работы с разреженными данными.
- Ускоряет процесс сходимости модели.

Возможные недостатки:
- Чувствительность к гиперпараметрам `ρ` и `ε`.
- В некоторых случаях может приводить к нестабильному процессу обучения.
