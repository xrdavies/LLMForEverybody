## 1. Функция GLU

![alt text](../../../01-第一章-预训练/assest/神经网络的激活函数/8.png)

GLU (Gated Linear Unit, управляемый линейный блок) — функция активации, которая используется в глубоком обучении для улучшения качества модели. GLU вводит gating-механизм, позволяя модели выборочно пропускать информацию и тем самым повышать выразительную способность и производительность.

### 1.1 Математическое определение

Математическое выражение функции GLU:

$$\text{GLU}(x) = (X ∗ W + b) ⊗ σ(X ∗ V + c)$$
где ⊗ обозначает поэлементное умножение, $X$ — вход, $W$ и $V$ — матрицы весов, $b$ и $c$ — смещения.

### 1.2 Ключевые свойства

1. **Gating-механизм**: GLU вводит gating-механизм, благодаря которому модель может выборочно пропускать информацию, повышая свою выразительную способность.
2. **Нелинейность**: GLU сочетает линейное преобразование и нелинейную активацию, позволяя модели изучать сложные паттерны.
3. **Фильтрация информации**: за счет gating-механизма GLU может отфильтровывать неважную информацию, усиливая качество модели.

### 1.3 Когда была предложена

Функция активации GLU была предложена в 2017 году Yann Dauphin и соавторами в статье «Language Modeling with Gated Convolutional Networks».

### 1.4 Какие проблемы решает

1. **Избирательность информации**: GLU использует gating-механизм, позволяя модели выборочно пропускать информацию и повышать выразительную способность.
2. **Усиление нелинейности**: GLU сочетает линейное преобразование и нелинейную активацию, усиливая нелинейные свойства модели.
3. **Повышение производительности модели**: GLU хорошо показывает себя во многих задачах, особенно в обработке естественного языка (NLP) и задачах моделирования последовательностей.


### 1.5 Пример

Ниже приведен простой пример на Python, показывающий, как вычислить функцию GLU:

```python
import numpy as np
def glu(x):
    """GLU 激活函数

    参数:
    x -- 输入数组，维度必须是偶数

    返回:
    GLU 激活后的数组
    """
    assert x.shape[-1] % 2 == 0, "输入数组的最后一个维度必须是偶数"
    half_dim = x.shape[-1] // 2
    return x[..., :half_dim] * sigmoid(x[..., half_dim:])

def sigmoid(x):
    """Sigmoid 函数

    参数:
    x -- 输入值

    返回:
    Sigmoid 函数的输出值
    """
    return 1 / (1 + np.exp(-x))
```

## 2. Функция Swish

![alt text](../../../01-第一章-预训练/assest/神经网络的激活函数/9.png)

Swish — широко применяемая в глубоком обучении функция активации, предложенная командой Google Brain. Swish вводит обучаемый параметр, благодаря которому функция активации может адаптивно настраиваться в процессе обучения и повышать качество модели.

### Математическое определение

Математическое выражение функции Swish:

$$\text{Swish}(x) = x \cdot \sigma(\beta x)$$

где:
- $x$ — вход.
- $\sigma$ — Sigmoid-функция, определяемая как $\sigma(x) = \frac{1}{1 + e^{-x}}$.
- $\beta$ — обучаемый параметр, управляющий формой функции.

В большинстве случаев $\beta$ задают равным 1, и выражение упрощается до:

$$\text{Swish}(x) = x \cdot \sigma(x)$$

### Ключевые свойства

1. **Гладкость**: функция Swish непрерывна и гладка, что помогает повысить стабильность модели и скорость сходимости.
2. **Немонотонность**: функция Swish немонотонна: на одних участках она возрастает, а на других убывает. Это свойство позволяет Swish улавливать более сложные паттерны.
3. **Обучаемость**: благодаря обучаемому параметру $\beta$ функция Swish может адаптивно настраиваться во время обучения, повышая качество модели.
4. **Приближение к ReLU**: когда $\beta$ стремится к бесконечности, функция Swish приближается к ReLU; когда $\beta$ стремится к 0, Swish приближается к линейной функции.

### Когда была предложена

Функция активации Swish была предложена в 2017 году командой Google Brain в статье «Searching for Activation Functions».

### Какие проблемы решает

1. **Гладкая активация**: Swish вводит Sigmoid-функцию, благодаря чему активация остается гладкой и на положительном, и на отрицательном участках входа, повышая стабильность модели.
2. **Немонотонность**: немонотонность Swish позволяет ей улавливать более сложные паттерны и делает ее выразительнее монотонных функций активации вроде ReLU.
3. **Обучаемость**: Swish вводит обучаемый параметр $\beta$, позволяя функции активации адаптивно настраиваться и улучшать качество модели.


### Пример

Ниже приведен простой пример на Python, показывающий, как вычислить функцию Swish:

```python
import numpy as np
def swish(x,beta=1.0):
    """Swish 激活函数

    参数:
    x -- 输入值

    返回:
    Swish 激活后的值
    """
    return x * sigmoid(beta*x)

def sigmoid(x):
    """Sigmoid 函数

    参数:
    x -- 输入值

    返回:
    Sigmoid 函数的输出值
    """
    return 1 / (1 + np.exp(-x))
```

## 3. Функция SwiGLU

SwiGLU (Swish-Gated Linear Unit) — функция активации, объединяющая свойства Swish и GLU (Gated Linear Unit) и предназначенная для повышения качества моделей глубокого обучения. За счет gating-механизма и функции активации Swish SwiGLU позволяет модели эффективнее выборочно пропускать информацию, усиливая выразительную способность и производительность.

### 3.1 Математическое определение

Математическое выражение функции SwiGLU:

$$\text{SwiGLU}(a, b) = \text{Swish}(a) \otimes \sigma(b)$$

где:
- $a$ и $b$ — входные тензоры.
- $\text{Swish}(x) = x \cdot \sigma(x)$ — функция активации Swish.
- $\sigma(x) = \frac{1}{1 + e^{-x}}$ — функция активации Sigmoid.
- $\otimes$ обозначает поэлементное умножение (Hadamard product).

### 3.2 Ключевые свойства

1. **Gating-механизм**: SwiGLU вводит gating-механизм, благодаря которому модель может выборочно пропускать информацию, повышая свою выразительную способность.
2. **Гладкость**: гладкость функции Swish помогает повысить стабильность модели и скорость сходимости.
3. **Немонотонность**: немонотонность функции Swish позволяет SwiGLU улавливать более сложные паттерны.
4. **Фильтрация информации**: за счет gating-механизма SwiGLU может отфильтровывать неважную информацию, усиливая качество модели.

### 3.3 Когда была предложена

Функция активации SwiGLU была предложена в 2021 году командой DeepMind в статье «Scaling Vision Transformers».

### 3.4 Какие проблемы решает

1. **Избирательность информации**: SwiGLU использует gating-механизм, позволяя модели выборочно пропускать информацию и повышать выразительную способность.
2. **Гладкая активация**: Swish вводит Sigmoid-функцию, благодаря чему активация остается гладкой и на положительном, и на отрицательном участках входа, повышая стабильность модели.
3. **Немонотонность**: немонотонность Swish позволяет SwiGLU улавливать более сложные паттерны и делает ее выразительнее монотонных функций активации вроде ReLU.

### 3.5 Пример

Ниже приведен простой пример на Python, показывающий, как вычислить функцию SwiGLU:
$$ f(X) = (X ∗ W + b) ⊗ Swish(X ∗ V + c) $$

```python
import numpy as np
def SwiGLU(x):
    """SwiGLU 激活函数

    参数:
    x -- 输入数组，维度必须是偶数

    返回:
    SwiGLU 激活后的数组
    """
    assert x.shape[-1] % 2 == 0, "输入数组的最后一个维度必须是偶数"
    half_dim = x.shape[-1] // 2
    return x[..., :half_dim] * swish(x[..., half_dim:])

def swish(x,beta=1.0):
    """Swish 激活函数

    参数:
    x -- 输入值

    返回:
    Swish 激活后的值
    """
    return x * sigmoid(beta*x)

def sigmoid(x):
    """Sigmoid 函数

    参数:
    x -- 输入值

    返回:
    Sigmoid 函数的输出值
    """
    return 1 / (1 + np.exp(-x))

```

### Итоги

Функция активации SwiGLU объединяет преимущества Swish и GLU. За счет gating-механизма и гладкой немонотонности она решает некоторые врожденные проблемы функций активации вроде ReLU. SwiGLU хорошо показывает себя во многих задачах глубокого обучения, особенно в компьютерном зрении и обработке естественного языка. Gating-механизм и гладкая активация помогают модели быстрее сходиться и достигать более высокой производительности.
