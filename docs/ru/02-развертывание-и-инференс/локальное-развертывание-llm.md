# За 10 минут: локальное приватное развертывание LLM

## Введение

Сегодня приватное локальное развертывание LLM уже не является задачей с высоким техническим порогом. Чаще всего это вопрос доступа к правильной информации. Следуя этому руководству, даже новичок сможет запустить LLM на своем компьютере.

Технические требования: уметь нажимать мышью и печатать. Нам понадобятся всего два приложения:

Ollama: https://ollama.com/

ChatBox: https://chatboxai.app/zh

## Ollama

`Ollama` - open-source инструмент для обслуживания больших языковых моделей (`LLM`). Он упрощает локальный запуск LLM и снижает порог входа. С его помощью разработчики, исследователи и энтузиасты могут быстро экспериментировать, управлять и разворачивать современные open-source LLM, включая `Qwen2`, `Llama3`, `Phi3`, `Gemma2` и другие. `Ollama` предоставляет простой и эффективный интерфейс для создания, запуска и управления моделями, а также богатую библиотеку готовых моделей для интеграции в разные приложения.

### 1. Скачать Ollama

Скачайте `Ollama` с официального сайта: https://ollama.com/

![alt text](../../../02-第二章-部署与推理/assest/10分钟私有化部署大模型到本地/1.png)

### 2. Установить Ollama

Пройдите обычный установщик: несколько кликов - и установка завершена.

### 3. Скачать модель

В библиотеке моделей `Ollama` по адресу https://ollama.com/library найдите подходящую модель. Для простого теста удобно выбрать небольшую `tiny`-модель.

![alt text](../../../02-第二章-部署与推理/assest/10分钟私有化部署大模型到本地/2.png)

Можно использовать модель `qwen2:0.5b`. Она применяет 4-bit квантизацию, а размер модели составляет всего 352 MB.

![alt text](../../../02-第二章-部署与推理/assest/10分钟私有化部署大模型到本地/3.png)

В терминале `cmd` введите команду для загрузки модели:

> ollama pull qwen2:0.5b

![alt text](../../../02-第二章-部署与推理/assest/10分钟私有化部署大模型到本地/4.png)

### 4. Запустить Ollama

Введите команду, чтобы запустить `qwen2:0.5b`:

> ollama run qwen2:0.5b

![alt text](../../../02-第二章-部署与推理/assest/10分钟私有化部署大模型到本地/5.png)

На этом этапе на вашем компьютере уже есть локальная LLM. При желании можно загрузить более крупную модель, чтобы получить лучшее качество ответов.

## ChatBox

`Chatbox AI` - клиентское AI-приложение и интеллектуальный ассистент. Оно поддерживает множество современных AI-моделей и API и доступно для `Windows`, `MacOS`, `Android`, `iOS`, `Linux` и веб-версии.

### 1. Скачать ChatBox

Скачайте `ChatBox` с официального сайта: https://chatboxai.app/zh#download

![alt text](../../../02-第二章-部署与推理/assest/10分钟私有化部署大模型到本地/6.png)

### 2. Установить ChatBox

Пройдите обычный установщик: несколько кликов - и установка завершена.

### 3. Настроить ChatBox

Найдите настройки в левом нижнем углу `ChatBox`.

![alt text](../../../02-第二章-部署与推理/assest/10分钟私有化部署大模型到本地/7.png)

В качестве поставщика модели выберите `ollama`.

![alt text](../../../02-第二章-部署与推理/assest/10分钟私有化部署大模型到本地/8.png)

Укажите API:

> http://localhost:11434

Укажите модель:

> qwen2:0.5b

Нажмите сохранить.

Проверьте работу.

![alt text](../../../02-第二章-部署与推理/assest/10分钟私有化部署大模型到本地/9.png)

Видно, что ответ уже сгенерирован через `qwen2:0.5b`. Если у вас мощный GPU, можно загрузить и развернуть более крупную модель, чтобы получить лучший результат.

Весь процесс не требует глубоких знаний компьютеров: достаточно уметь устанавливать обычные приложения.
